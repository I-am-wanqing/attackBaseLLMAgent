---
title: "Terminal"
description: "用于运行命令和安全工具的 Bash Shell"
---

Strix 可以访问运行在 Docker 沙箱中的持久化 bash 终端，从而让智能体能够使用所有[预装安全工具](/tools/sandbox)。

## 能力

| 功能 | 说明 |
| ----------------- | ---------------------------------------------------------- |
| 持久状态 | 工作目录和环境变量会在命令之间保留 |
| 多会话 | 并行运行多个终端以执行并发操作 |
| 后台任务 | 启动长时间运行的进程而不阻塞 |
| 交互式 | 响应提示并控制正在运行的进程 |

## 常见用途

### 运行安全工具

```bash
# 子域枚举
subfinder -d example.com

# 漏洞扫描
nuclei -u https://example.com

# SQL 注入测试
sqlmap -u "https://example.com/page?id=1"
```

### 代码分析

```bash
# 快速 SAST 初筛
semgrep --config auto ./src

# 结构化 AST 搜索
sg scan ./src

# 密钥检测
gitleaks detect --source ./
trufflehog filesystem ./

# 供应链与配置错误检查
trivy fs ./
```

### 自定义脚本

```bash
# 运行 Python 利用脚本
python3 exploit.py

# 执行 shell 脚本
./test_auth_bypass.sh
```

## 会话管理

智能体可以并发运行多个终端会话，例如：

- 主会话用于主要测试
- 副会话用于监控
- 后台进程用于服务或监视器
