Strix 智能体使用专门工具，以像真实渗透测试人员一样测试你的应用。

## 核心工具

<CardGroup cols={2}>
  <Card title="Browser" icon="globe" href="/tools/browser">
    由 Playwright 驱动的 Chrome，用于交互 Web 界面。
  </Card>
  <Card title="HTTP Proxy" icon="network-wired" href="/tools/proxy">
    由 Caido 驱动的代理，用于拦截和重放请求。
  </Card>
  <Card title="Terminal" icon="terminal" href="/tools/terminal">
    用于运行命令和安全工具的 Bash shell。
  </Card>
  <Card title="Sandbox Tools" icon="toolbox" href="/tools/sandbox">
    预装安全工具：Nuclei、ffuf 等。
  </Card>
</CardGroup>

## 附加工具

| Tool           | Purpose                                  |
| -------------- | ---------------------------------------- |
| Python Runtime | 编写并执行自定义利用脚本 |
| File Editor    | 读取并修改源代码              |
| Web Search     | 通过 Perplexity 进行实时 OSINT           |
| Notes          | 在扫描过程中记录发现        |
| Reporting      | 生成包含 PoC 的漏洞报告 |
