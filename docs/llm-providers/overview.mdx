Strix 使用 [LiteLLM](https://docs.litellm.ai/docs/providers) 以兼容模型，支持 100+ 个 LLM 提供方。

## 配置

设置你的模型和 API 密钥：

| 模型             | 提供方      | 配置                    |
| ----------------- | ------------- | -------------------------------- |
| GPT-5.4           | OpenAI        | `openai/gpt-5.4`                 |
| Claude Sonnet 4.6 | Anthropic     | `anthropic/claude-sonnet-4-6`    |
| Gemini 3 Pro      | Google Vertex | `vertex_ai/gemini-3-pro-preview` |

```bash
export STRIX_LLM="openai/gpt-5.4"
export LLM_API_KEY="your-api-key"
```

## 本地模型

可通过 [Ollama](https://ollama.com)、[LM Studio](https://lmstudio.ai) 或任意兼容 OpenAI 的服务在本地运行模型：

```bash
export STRIX_LLM="ollama/llama4"
export LLM_API_BASE="http://localhost:11434"
```

有关设置说明和推荐模型，请参阅 [本地模型指南](/llm-providers/local)。

## 模型格式

使用 LiteLLM 的 `provider/model-name` 格式：

```
openai/gpt-5.4
anthropic/claude-sonnet-4-6
vertex_ai/gemini-3-pro-preview
bedrock/anthropic.claude-4-5-sonnet-20251022-v1:0
ollama/llama4
```
