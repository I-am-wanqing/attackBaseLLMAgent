## 配置

```bash
export STRIX_LLM="openai/gpt-5.4"
export LLM_API_KEY="sk-..."
```

## 可用模型

完整可用模型列表请参阅 [OpenAI 模型文档](https://platform.openai.com/docs/models)。

## 获取 API 密钥

1. 前往 [platform.openai.com](https://platform.openai.com)
2. 进入 API Keys
3. 创建新的 secret key

## 自定义 Base URL

适用于 OpenAI 兼容 API：

```bash
export STRIX_LLM="openai/gpt-5.4"
export LLM_API_KEY="your-key"
export LLM_API_BASE="https://your-proxy.com/v1"
```
