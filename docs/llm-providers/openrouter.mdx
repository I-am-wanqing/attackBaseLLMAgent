---

[OpenRouter](https://openrouter.ai) 通过单一 API 提供来自多个提供方的 100+ 个模型。

## 配置

```bash
export STRIX_LLM="openrouter/openai/gpt-5.4"
export LLM_API_KEY="sk-or-..."
```

## 可用模型

使用 `openrouter/<provider>/<model>` 格式即可访问 OpenRouter 上的任意模型：

| 模型 | 配置 |
|-------|---------------|
| GPT-5.4 | `openrouter/openai/gpt-5.4` |
| Claude Sonnet 4.6 | `openrouter/anthropic/claude-sonnet-4.6` |
| Gemini 3 Pro | `openrouter/google/gemini-3-pro-preview` |
| GLM-4.7 | `openrouter/z-ai/glm-4.7` |

## 获取 API 密钥

1. 前往 [openrouter.ai](https://openrouter.ai)
2. 登录并进入 Keys
3. 创建新的 API key

## 优势

- **单一 API** — 访问 OpenAI、Anthropic、Google、Meta 等来源的模型
- **故障切换路由** — 在不同提供方之间自动切换
- **成本跟踪** — 监控所有模型的使用情况
- **更高限流** — OpenRouter 会帮你处理提供方限制
