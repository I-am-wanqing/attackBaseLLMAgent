## 安装

Vertex AI 需要 Google 云平台依赖。请通过 vertex 额外依赖安装 Strix：

```bash
pipx install "strix-agent[vertex]"
```

## 配置

```bash
export STRIX_LLM="vertex_ai/gemini-3-pro-preview"
```

无需 API 密钥，使用 Google 云平台的 Application Default Credentials。

## 认证

### 方案 1：gcloud CLI

```bash
gcloud auth application-default login
```

### 方案 2：服务账号

```bash
export GOOGLE_APPLICATION_CREDENTIALS="/path/to/service-account.json"
```

## 可用模型

| 模型 | 说明 |
|-------|-------------|
| `vertex_ai/gemini-3-pro-preview` | 安全测试的整体最佳表现 |
| `vertex_ai/gemini-3-flash-preview` | 更快且更便宜 |

## 项目配置

```bash
export VERTEXAI_PROJECT="your-project-id"
export VERTEXAI_LOCATION="global"
```

## 先决条件

1. 在你的 Google 云平台项目中启用 Vertex AI API
2. 确保你的账户拥有 `Vertex AI User` 角色
